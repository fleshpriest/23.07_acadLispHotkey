## What

This repo contains a collection of scripts & utilities that I've developed for helping me engineer in AutoCAD.


## How

LISP to automate routines with AutoCAD.
[AutoHotkey](https://www.autohotkey.com/) to trigger the LISP commands with minimal keystrokes.


## Why

To work more efficiently.
The aim is to make commands within AutoCAD happen as close to the speed of thought as possible.


## Other details

These scripts will not be fully plug-and-play for others. Of note,
- The AutoHotkey script has been designed for custom keyboards using custom keymaps.
- Any LISP commands featuring 'insert' instructions require an extension to the AutoCAD path & are intended to be used with my block library (not included at this time.)
- Certain filenames, paths, & other details have been omitted completely or replaced with placeholders.
- The AutoHotkey script is written for the older version 1.1
